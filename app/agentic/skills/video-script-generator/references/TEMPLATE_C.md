# Type C: 缺口模板 (Critical Gap)

## 一句话定义

讲一个关键缺口：缺口是什么 → 不补会怎样 → 最低门槛是什么 → 补齐方向（不教学）→ 引导领"尺子卡片"对照。

## 适用场景

- 强调某个关键要求/门槛
- "一票否决"类内容
- 必须具备的条件/资质
- 警示性、提醒性内容

## 5条固定槽位（必须按顺序）

### 1. 缺口怎么定义
**内容：** 用大白话定义缺口（1句）

**示例：**
- "什么叫主体缺口？就是对方公司是个空壳，没有实际资产，签了合同也拿不到钱"
- "什么叫确权缺口？就是对方没有书面确认你干了活、交付了东西，到时他说没干，你拿不出证据"
- "什么叫执行缺口？就是你赢了官司，但对方账户没钱、房产不是他的，执行不了"
- "什么叫交付缺口？就是你东西发了、活干了，但对方没验收，到时他说没收到/不合格"

**写作要点：**
- 用"什么叫...？就是..."句式
- 要用大白话，不用专业术语
- 要让外行人一听就懂
- 定义要具体，不能模糊

---

### 2. 不补的后果
**内容：** 会导致什么白忙/推进慢（1句）

**示例：**
- "不补这个缺口，合同签了也是废纸，钱还是要不回来"
- "不补这个缺口，对方到时翻脸不认账，你连起诉的证据都不够"
- "不补这个缺口，赢了官司也拿不到钱，还要倒贴律师费和诉讼费"
- "不补这个缺口，对方一直拖着不验收，你的尾款永远拿不到"

**写作要点：**
- 用"不补这个缺口..."开头
- 要说清楚后果，要吓到用户
- 后果要具体（白忙/倒贴钱/拿不到钱）
- 不能委婉，要直接警示

---

### 3. 最低补齐门槛
**内容：** 至少补到什么程度才值得继续（1句）

**示例：**
- "最低门槛：对方账户里要有现金，或者名下有房产、车辆能查封"
- "最低门槛：要有对方盖章的验收单，或者法定代表人签字的确认函"
- "最低门槛：对方要书面承诺付款时间，并且至少付了一部分款"
- "最低门槛：合同要写清楚验收标准，并且对方签字确认了"

**写作要点：**
- 用"最低门槛："开头
- 要给出明确的"及格线"
- 不能要求太高，要现实
- 要可判断、可验证

---

### 4. 补齐方向
**内容：** 往哪个方向补（不写步骤，1句）

**示例：**
- "补齐方向：先查对方公司的资产情况，再看要不要继续合作"
- "补齐方向：让对方补盖公章，或者让法定代表人签字确认"
- "补齐方向：把口头承诺变成书面协议，并且让对方先付一部分"
- "补齐方向：把验收标准写清楚，并且让对方签字确认后再开工"

**写作要点：**
- 用"补齐方向："开头
- **只写方向，不写步骤**
- 不要写成"第一步...第二步..."
- 要给方向，不给教程

---

### 5. 自查提示 + 动作
**内容：** 一句自查提示 + 私信【评估】领《回款判断尺子卡》

**示例：**
- "自查一下：你合作的对方，账户里有没有钱？私信【评估】，领《回款判断尺子卡》，对照看看你缺不缺这一条"
- "自查一下：你的验收单上，有没有对方公章？私信【评估】，领《回款判断尺子卡》，看看你缺的是哪3条"
- "自查一下：你的合同里，有没有验收标准？私信【评估】，领《回款判断尺子卡》，对照看看你缺不缺这一条"

**写作要点：**
- 自查提示要与缺口直接相关
- 要能用"是/否"回答
- 动作要统一：私信【评估】领《回款判断尺子卡》
- 要强调"对照""自查"

---

## 一句收口口径

**Type C 用"模板"给用户一把尺子：**
"先对照门槛，看你缺不缺这一条。"

## 完整示例

### 示例1：主体缺口

**(开头钩子)**
"回款最怕什么？不是对方不付，而是对方没能力付。"

**(5个槽位)**
"什么叫主体缺口？就是对方公司是个空壳，没有实际资产，签了合同也拿不到钱。
不补这个缺口，合同签了也是废纸，起诉了也拿不到钱，还要倒贴律师费和诉讼费。
最低门槛：对方账户里要有现金，或者名下有房产、车辆能查封。
补齐方向：先查对方公司的资产情况，再看要不要继续合作。
自查一下：你合作的对方，账户里有没有钱？如果账户余额是0，别指望了。"

**(CTA)**
"私信【评估】，领《回款判断尺子卡》，对照看看你缺不缺这一条。"

**(收口)**
"先对照门槛，看你缺不缺这一条。"

---

### 示例2：确权缺口

**(开头钩子)**
"为什么你的尾款要不到？可能你连'对方承认欠钱'的证据都没有。"

**(5个槽位)**
"什么叫确权缺口？就是对方没有书面确认你干了活、交付了东西，到时他说没干，你拿不出证据。
不补这个缺口，对方到时翻脸不认账，你连起诉的证据都不够，法官可能直接驳回你的案子。
最低门槛：要有对方盖章的验收单，或者法定代表人签字的确认函。
补齐方向：让对方补盖公章，或者让法定代表人签字确认。
自查一下：你的验收单上，有没有对方公章？如果只有项目经理签字，风险很大。"

**(CTA)**
"私信【评估】，领《回款判断尺子卡》，对照看看你缺不缺这一条。"

**(收口)**
"先对照门槛，看你缺不缺这一条。"

---

### 示例3：交付缺口

**(开头钩子)**
"你活干了、货发了，为什么尾款还是要不到？"

**(5个槽位)**
"什么叫交付缺口？就是你东西发了、活干了，但对方没验收，到时他说没收到/不合格，你没法反驳。
不补这个缺口，对方一直拖着不验收，你的尾款永远拿不到，而且时间越久越难要。
最低门槛：合同要写清楚验收标准，并且对方签字确认了。
补齐方向：把验收标准写清楚，并且让对方签字确认后再开工。
自查一下：你的合同里，有没有验收标准？如果只写了'验收合格后付款'，等于没写。"

**(CTA)**
"私信【评估】，领《回款判断尺子卡》，对照看看你缺不缺这一条。"

**(收口)**
"先对照门槛，看你缺不缺这一条。"

## 写作检查清单

- [ ] 缺口定义是否用了"什么叫...？就是..."？
- [ ] 定义是否用大白话，外行人能听懂？
- [ ] 后果是否具体、有警示性？
- [ ] 最低门槛是否明确、可验证？
- [ ] 补齐方向是否只写了方向，没写步骤？
- [ ] 自查提示是否与缺口直接相关？
- [ ] CTA是否统一指向《回款判断尺子卡》？
- [ ] 收口口径是否用了"先对照门槛，看你缺不缺这一条"？
- [ ] 全文是否口语化？是否有"警示感"？

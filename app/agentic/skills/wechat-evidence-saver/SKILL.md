# 微信证据保存指南

## 快速开始

当用户问"微信聊天记录能当证据吗"或"怎么保存微信证据"时：

### Step 1: 确认证据类型

微信记录属于**电子数据**，根据内容分6类：

| 类型 | 示例 |
|------|------|
| 文字记录 | 微信聊天内容、借条文字 |
| 图片记录 | 聊天图片、截图 |
| 语音记录 | 微信语音消息 |
| 视频记录 | 微信视频消息 |
| 网络链接 | 分享的链接 |
| 转账信息 | 微信红包、转账记录 |

### Step 2: 指导保存原始证据

**关键原则**：保留原始载体（手机/电脑）

**必须做**：
1. **不要删除聊天记录**
2. **不要使用"一键清理"功能**
3. **保存原始载体**

**提交证据时需提供**：
1. 登录微信的过程演示（证明持有账户）
2. 双方个人信息界面（微信号、昵称、头像）
3. 完整的聊天记录（不能选择性提供）

### Step 3: 法庭展示步骤

当法官要求展示微信证据时：

1. 账户持有人登录微信
2. 展示登录所使用的账户名称
3. 在通讯录中找到对方，查看个人信息
4. 进入对话，逐条展示内容
5. 展示转账记录：聊天详情 → 查看转账记录

### Step 4: 证据保全选项

**推荐程度排序**：

| 方式 | 证明力 | 费用 |
|------|--------|------|
| 公证处公证 | 高 | 中 |
| 第三方存证平台 | 高 | 低 |
| 自己保存原始载体 | 中 | 无 |

**建议**：
- 重要证据→公证处公证
- 一般证据→保留原始载体

### Step 5: 调取微信转账记录

用户可自行调取，不需法院调取：

**操作步骤**：
1. 我 → 支付 → 钱包 → 账单
2. 点击"常见问题" → "下载账单"
3. 选择"用做证明材料"或"用于个人对账"
4. 选择日期区间（最长一年）

## 特殊注意事项

### 微信语音
- 不能单独作为定案依据
- 需保存原始记录
- 内容须客观真实、连贯
- 双方有明确表态
- 需其他证据佐证

### 微信图片
- 需与聊天记录整体公证
- 不要单独对图片保全

### 微信视频
- 自行拍摄的视频证明力更强
- 转载/制作视频证明力较弱

### 转账记录
- 保留原始转账记录
- 注明转账用途
- 辅助电话录音、短信等证据

## 常见问题

### Q：聊天记录删了能恢复吗？
A：iPhone可尝试：微信 → +号 → 添加朋友 → 搜索`:recover`
Android需用专业软件

### Q：截屏能当证据吗？
A：单独截屏证明力弱，需与原始载体核对

### Q：对方不承认微信号是他的怎么办？
A：结合微信号不可更改特点 + 实名认证信息 + 手机号等多维度验证

# 证据类型识别

## 快速开始

当用户问"XX能当证据吗"或"这是什么类型的证据"时：

### Step 1: 识别证据类型

根据材料特征判断属于哪一类：

| 材料类型 | 特征 | 证据类型 |
|----------|------|----------|
| 合同书、借条、收据 | 文字内容 | 书证 |
| 实物样品、争议标的物 | 实物形态 | 物证 |
| 录音、录像 | 音视频 | 视听资料 |
| 微信记录、短信、邮件 | 电子形式 | 电子数据 |
| 证人陈述 | 第三人作证 | 证人证言 |
| 鉴定报告 | 专业鉴定 | 鉴定意见 |
| 现场勘查记录 | 法院勘验 | 勘验笔录 |
| 当事人陈述 | 当事人所说 | 当事人陈述 |

### Step 2: 说明收集要求

根据证据类型，告诉用户：

**书证**
- 尽量提交原件
- 复印件需说明为什么无法提交原件
- 外文书证必须附中文译本

**物证**
- 提交原物
- 提交原物困难的可提交照片

**视听资料**
- 保证完整性，不要剪辑
- 原始载体保存好

**电子数据**
- 微信/短信：提供完整对话截图或导出记录
- 邮件：提供原始邮件或带时间戳的导出件
- 网页：保全截图或公证

**证人证言**
- 证人需出庭作证
- 证人需了解案件事实

**鉴定意见**
- 需由有资质机构出具
- 说明鉴定依据和方法

### Step 3: 提醒注意事项

根据类型提醒关键点：

**电子数据特别提醒**
- 原始载体（手机/电脑）保管好
- 核对微信号/手机号实名认证
- 完整截图，不要只截对自己有利的部分

**视听资料特别提醒**
- 录音录像需未剪辑
- 涉及隐私的证据质证时不公开审理

**单位证明特别提醒**
- 需单位负责人签名 + 经办人签名 + 单位盖章
- 三者缺一不可

## 快速对照表

| 用户问 | 识别 | 回复要点 |
|--------|------|----------|
| "微信聊天记录能当证据吗" | 电子数据 | 提供完整原始记录，核对实名 |
| "录音能当证据吗" | 视听资料 | 原始载体保存，未剪辑 |
| "合同原件丢了" | 书证 | 复印件可用，但需说明原因 |
| "找证人作证" | 证人证言 | 证人需出庭了解案情 |
| "对方不承认怎么办" | 鉴定意见 | 可申请笔迹/印章鉴定 |

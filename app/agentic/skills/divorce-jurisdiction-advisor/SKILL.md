# 离婚管辖法院确定

## 快速开始

当用户问"离婚在哪起诉"或"怎么确定离婚法院"时：

### Step 1: 确定基本原则

**一般原则**：原告就被告
- 由被告住所地人民法院管辖
- 被告住所地与经常居住地不一致的 → 经常居住地法院

**例外原则**：原告所在地法院
- 被告下落不明/被宣告失踪
- 被告被采取强制性教育措施
- 被告被监禁

### Step 2: 特殊情形处理

| 情形 | 管辖法院 | 注意事项 |
|------|----------|----------|
| **双方都同意离婚** | 可协议选择 | 需双方书面协议 |
| **一方在境外** | 境内一方住所地法院 | 境外一方可能需委托代理人 |
| **一方被监禁** | 原告住所地法院 | 无论刑期长短 |
| **一方下落不明** | 原告住所地法院 | 需先申请宣告失踪 |
| **军人离婚** | 对方/非军人一方所在地法院 | 特殊规定需注意 |

### Step 3: 收集管辖证明材料

**被告住所地证据**：
- 被告身份证信息
- 户籍证明

**经常居住地证据**（如需在原告经常居住地起诉）：
- 居住证/暂住证
- 租房合同或房产证
- 一年以上水电费/物业费缴纳凭证
- 社区/居委会证明

### Step 4: 常见问题处理

**问题1：对方不在户籍地**
- 查询被告经常居住地
- 收集经常居住地证明材料
- 向经常居住地法院起诉

**问题2：对方下落不明**
- 先向法院申请宣告失踪
- 凭宣告失踪判决在原告住所地起诉
- 或直接起诉，法院公告送达

**问题3：如何申请调查令**
- 申请法院调查被告户籍/居住信息
- 委托律师查询
- 向公安机关申请人口信息查询

## 离婚诉讼其他注意事项

### 准备材料清单

| 材料 | 说明 |
|------|------|
| 起诉状 | 写明诉讼请求、事实理由 |
| 身份证明 | 身份证、结婚证 |
| 证据材料 | 感情破裂证据、财产证据等 |
| 管辖证据 | 如需在原告经常居住地起诉 |

### 诉讼策略建议

1. **先尝试协商**：节省时间和费用
2. **选择有利法院**：考虑证据便利、执行便利
3. **准备充分证据**：感情破裂、财产状况、子女抚养
4. **考虑管辖异议**：如对方先起诉，可提管辖异议

## 关联法规速查

| 问题 | 法条依据 |
|------|----------|
| 一般管辖 | 民事诉讼法第22条 |
| 原告住所地管辖 | 民事诉讼法第23条 |
| 经常居住地定义 | 民诉法解释第4条 |
| 监禁人员管辖 | 民诉法解释第8条 |
| 下落不明管辖 | 民诉法解释第9条 |

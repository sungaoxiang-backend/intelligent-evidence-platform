# 经常居住地证明

## 快速开始

当用户问"在哪起诉"或"怎么证明经常居住地"时：

### Step 1: 区分概念

**住所地** = 户籍所在地（身份证地址）
- 简单：复印身份证即可

**经常居住地** = 离开户籍地至起诉时连续居住满一年的地方
- 复杂：需要提供多种证明材料

### Step 2: 确认构成要件

问用户三个问题：

1. **空间要件**：在户籍地以外的地方居住？
2. **时间要件**：已连续居住满一年以上？
3. **目的要件**：非因住院就医而居住？

三个要件都满足 → 可以主张经常居住地管辖

### Step 3: 收集证明材料

根据用户情况，指导收集：

| 情况 | 所需材料 |
|------|----------|
| **有居住证** | 居住证 + 居住证流水（显示连续一年记录） |
| **租房居住** | 租赁合同 + 一年以上房租支付凭证 + 水电费凭证 |
| **购房居住** | 房产证 + 一年以上水电费凭证 + 物业费缴纳凭证 |
| **公司宿舍** | 社保缴纳证明 + 公司居住证明 |
| **无正式证明** | 水电煤气缴费记录（一年以上）+ 参考入库案例 |

### Step 4: 证明力排序

国家机关出具 > 单位出具 > 个人出具

**建议**：尽可能收集多种类型、相互印证的证据

## 常见场景

### 场景：原告想在自己居住地起诉

问用户：
1. 在现住地住了多久？（需满一年）
2. 有居住证吗？
3. 有租房合同或房产证吗？

根据回答，指导准备对应材料。

### 场景：材料被立案庭退回

可能原因：
- 材料不完整
- 证据之间无法相互印证
- 未达到"一年以上"要求

建议：
- 补充水电费缴费记录
- 联系物业/居委会开证明
- 参考入库案例说明

## 证据类型速查

| 证据类型 | 证明力 | 获取难度 |
|----------|--------|----------|
| 居住证/暂住证 | 高 | 中 |
| 社保缴纳证明 | 高 | 低 |
| 物业居住证明 | 中 | 低 |
| 居委会/村委会证明 | 中 | 低 |
| 租房合同 + 房租凭证 | 中 | 低 |
| 水电费缴费记录 | 中 | 低 |
| 单位工作证明 | 中 | 低 |

# 主张债权起诉书写作指南

一、法条引用与分析
根据《中华人民共和国合同法》百零七条规定：“当事人一方不履行合同义务或者不完全履行合同义务，应当承担违约责任。”这一规定明确了债务人的法律责任，为债权人提供了法律依据。

二、起诉书的撰写要点
主张债权的起诉书应当包含以下要素：

1.双方当事人的基本信息；

2.合同关系及债权债务事实的详细描述；

3.违约行为的明确指出；

4.具体的诉讼请求，包括债权本金、利息、违约金等；

5.相关证据的列举，如合同文本、欠条、转账记录等；

6.适用的法律条文和理由。

三、法律文书的格式与语言
起诉书应当使用规范的法律文书格式，语言要准确、简洁、明了。避免使用模糊不清或感情色彩的词汇，确保文书的专业性和严肃性。
